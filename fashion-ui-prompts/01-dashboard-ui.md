# Enterprise Fashion ERP/POS
# Executive Dashboard UI Generation

Design the production-ready Executive Dashboard UI for the Enterprise Fashion ERP/POS platform.

This is the UI Generation phase.

The Dashboard specification, business requirements, user flows, KPIs, information architecture, and design system have already been approved.

Do NOT redesign the dashboard.

Transform the approved specification into a production-ready enterprise SaaS interface.

--------------------------------------------------
USE PREVIOUS SPECIFICATIONS
--------------------------------------------------

Always inherit:

• Master Context

• Master System

• Design System

• Navigation

• Dashboard Specification

• Design Tokens

• UX State Management

Never modify business logic.

Never change navigation.

Never change KPIs.

Never invent new dashboard widgets unless required.

Reuse all existing enterprise components.
